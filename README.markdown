`cloudi_api_ocaml`
==================

OCaml [CloudI API](https://cloudi.org/api.html#1_Intro)

Build
-----

    opam pin add cloudi . -n
    opam install cloudi --verbose
    opam remove cloudi --verbose
    opam pin remove cloudi
